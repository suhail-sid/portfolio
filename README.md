## Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* ... */
}
```

### Content
- Update personal information in `index.html`
- Modify project details in the Projects section
- Add/remove skills in the Skills section

### 3D Particles
Adjust particle settings in `script.js`:
```javascript
const particleCount = 2000; // Number of particles
```
